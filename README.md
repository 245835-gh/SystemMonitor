АНОТАЦІЯ
===============================================================================

Програмний засіб призначений для моніторингу апаратних ресурсів комп'ютера в
режимі реального часу. Дозволяє відстежувати завантаження центрального
процесора, оперативної пам'яті, дисків, відеокарти, а також виконувати
сканування локальної мережі та керувати процесами операційної системи.

Основні можливості:
  - Моніторинг CPU та RAM у реальному часі з графіками історії
  - Перегляд, пошук та завершення процесів
  - Аналіз дисків та пошук найбільших папок
  - Сканування локальної мережі (Ping, порти, MAC-адреси, визначення ОС)
  - Моніторинг температури компонентів (CPU, GPU, HDD)
  - Інформація про відеокарту
  - Системна інформація (ОС, процесор, RAM, диски)
  - Список встановлених програм
  - Підтримка трьох тем оформлення (Dark, Light, Emerald)
  - Збереження налаштувань та результатів у JSON/TXT


ВИМОГИ ДО СИСТЕМИ
===============================================================================

  - Операційна система: Windows 10 / Windows 11 (64-bit)
  - Архітектура: x64
  - Пам'ять: 2 GB RAM (рекомендовано)
  - .NET Framework: версія 4.8 або вище
  - Процесор: 1 GHz або вище

ВСТАНОВЛЕННЯ ТА ЗАПУСК
===============================================================================

Портативна версія:
  1. Запустіть файл SystemMonitor.exe
  2. Програма не потребує встановлення

Інсталяційна версія:
  1. Запустіть файл SystemMonitorSetup.msi
  2. Дотримуйтесь інструкцій майстра встановлення
  3. Після встановлення програма з'явиться в меню "Пуск"
